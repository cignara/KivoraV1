### Register Modal (`#page-register`)
Multi-step sign-up flow for new accounts.
- Step 1: Account type (Parent / Teacher / Try as Guest)
- Step 2: Email + password
- Step 3: Child profile(s) — name, age, grade, language preference (English / Sinhala / Tamil)
- Step 4: Curriculum preference (USA / UK)

---

### Child Dashboard (`#page-child-dashboard`)
Sections inside the child-facing UI:
- 🚀 **Continue Learning** — Resume in-progress activities
- 🎯 **Today's Challenges** — Daily goals with XP rewards
- 🌍 **Explore All 9 Worlds** — World selector grid (all 9 visible and accessible)
- 🏆 **My Rewards** — Stars · XP · Gems · Coins · Badges · Certificates
- 🦊 **Kivi's Message** — Personalised narrated greeting

---

### Parent Dashboard Modal (`#page-parent-dashboard`)
Full in-app parent control panel including:
- Progress tracking per child across all 9 worlds
- Curriculum alignment view (Common Core / UK National Curriculum)
- Screen time settings
- **Bedtime Lock** — Set a time input (default: `20:00`) to block access after bedtime (triggers Kivi sleep animation)
- Weekly report preferences
- Language settings per child (English / Sinhala / Tamil)
