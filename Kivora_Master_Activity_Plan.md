## Global Kivi Voice Rule (Confirmed — All Content)

Every child-facing content item requires three speech scripts:

| Script | Purpose | Rule |
|---|---|---|
| `speakLoad` | Kivi reads the question/prompt when content loads | Required on every item |
| `speakOk` | Kivi celebrates the correct answer + reinforces the learning | Required on every item |
| `speakNo` | Kivi explains why the wrong answer was wrong — **must count, demonstrate, and teach** | Required — never just 'Try again!' |

Implementation: Web Speech API (built-in all browsers, no API key, works offline, voice auto-selected by device).
Rate: 0.86 · Pitch: 1.1 · Priority voices: Samantha (iOS) → Google US English → Aria/Zira (Edge) → en-US → en.
